# File Manager Project

A simple and efficient file handling application built with Python. This project demonstrates basic File I/O (Input/Output) operations and features a user-friendly interface.

## Features
- **CRUD Operations:** Create, Read, Update, and Delete files effortlessly.
- **Python-based:** Built using Python's built-in file handling capabilities.
- **Minimalist UI:** Designed for a clean, user-friendly experience.

## How to Run
1. Ensure you have [Python](https://www.python.org/) installed on your system.
2. Clone this repository:
```bash
git clone https://github.com/Anadi18-hub/file-manager-project.git
```
